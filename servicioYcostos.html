Mediciones de terreno

Servicio profesional para determinar con precisión las dimensiones de terrenos y resolver necesidades catastrales.

Costo: $7,000 MXN - $30,000 MXN
Tiempo: 1-5 días según complejidad
Incluye: Informe técnico + plano digital

Descripción completa

Medir un terreno tiene como propósito principal determinar con precisión sus dimensiones. Existen muchas razones por las que podría ser necesario o conveniente realizar esta medición:

  • Aclarar discrepancias catastrales
  • Llevar a cabo la división de propiedades
  • Resolver conflictos con dueños de terrenos vecinos
  • Preparación para proyectos de construcción

Proceso técnico

Un levantamiento topográfico consiste en medir y representar las características físicas de un terreno para crear un plano detallado que incluya:

  • Dimensiones exactas
  • Formas del terreno
  • Elevaciones
  • Elementos naturales o artificiales relevantes

Herramientas utilizadas

  • Teodolito: Para medir ángulos horizontales y verticales
  • Estación total: Combina teodolito con medidor electrónico de distancias
  • Drones: Para fotogrametría y mapas 3D
  • GPS de alta precisión: Para terrenos extensos

Planimetría topográfica

Representación en plano de las posiciones horizontales de los elementos de un terreno.

Costo: $5,000 MXN - $18,000 MXN
Entrega: Plano 2D en formato DWG + PDF
Escalas comunes: 1:100, 1:200, 1:500

¿Qué es la planimetría?

Rama de la topografía que representa en un plano las posiciones horizontales de los elementos de un terreno sin considerar su altura o relieve. Muestra la distribución de:

  • Límites de propiedades
  • Edificaciones existentes
  • Caminos y vías
  • Ríos o cuerpos de agua
  • Infraestructura pública

Aplicaciones principales

  • Proyectos de construcción civil
  • Urbanización y planificación municipal
  • Regularización catastral
  • Diseño de redes de servicios (agua, electricidad)

Diferencia con altimetría

Mientras la planimetría solo considera la posición horizontal, la altimetría se encarga de medir y representar las diferencias de elevación del terreno. Ambos servicios pueden combinarse para un levantamiento topográfico completo.

Doble perfil topográfico

Técnica para representar el terreno en dos cortes perpendiculares entre sí.

Costo: $8,000 MXN - $25,000 MXN
Entrega: Planos longitudinal + transversal
Usos: Diseño de vías, drenajes, urbanizaciones

¿Qué es el doble perfil?

Técnica que representa el terreno en dos cortes ortogonales:

  1. Perfil longitudinal: Siguiendo el eje principal del proyecto
  2. Perfil transversal: Mostrando las variaciones laterales

Aplicaciones clave

  • Diseño de carreteras: Para calcular movimientos de tierra
  • Sistemas de drenaje: Determinar pendientes óptimas
  • Urbanizaciones: Planificar niveles de pavimentación
  • Proyectos ferroviarios: Evitar pendientes peligrosas

Ventajas

  • Permite visualizar cambios bruscos de elevación
  • Identifica puntos críticos para excavación/relleno
  • Esencial para cálculos de volúmenes de tierra
  • Base para diseño de sistemas de drenaje pluvial

Curvas de nivel (P/C)

Representación de elevaciones mediante líneas que unen puntos de igual altura.

Costo: $6,500 MXN - $22,000 MXN
Equidistancia: 0.5m, 1m o personalizada
Formato: Plano impreso + digital (DWG)

¿Qué son las curvas de nivel?

Líneas imaginarias que conectan puntos de igual elevación sobre el nivel del mar. Cada curva representa una altura específica.

Interpretación

  • Curvas muy juntas: Pendiente pronunciada
  • Curvas separadas: Terreno plano o suave
  • P/C (Punto de Curvatura): Donde cambia la pendiente

Aplicaciones

  • Ingeniería civil: Diseño de cortes y rellenos
  • Arquitectura: Adaptación al terreno
  • Agricultura: Planificación de drenajes
  • Medio ambiente: Estudios de erosión

Detalles técnicos

Realizamos curvas de nivel con equidistancias desde 0.25m hasta 5m según requerimiento del proyecto. Utilizamos:

  • Estaciones totales de última generación
  • Drones para áreas extensas
  • Software especializado (AutoCAD Civil 3D, QGIS)

Plataformeo

Nivelación profesional de terrenos para preparación de obras.

Costo: $15,000 MXN - $50,000 MXN
Superficie mínima: 500 m²
Incluye: Topografía inicial + replanteo

¿Qué es el plataformeo?

Proceso de nivelación y compactación de terrenos para crear superficies planas estables, esencial para:

  • Cimentación de edificios
  • Pavimentación de calles
  • Instalación de plantas industriales
  • Construcción de pistas deportivas

Nuestro proceso

  1. Estudio topográfico inicial (curvas de nivel)
  2. Diseño de plataforma (niveles objetivo)
  3. Replanteo en campo (marcación de niveles)
  4. Supervisión de movimientos de tierra
  5. Control de compactación

Equipos utilizados

  • Niveladoras láser para alta precisión
  • Compactadores vibratorios
  • Estaciones totales para control geométrico
  • Drones para seguimiento de avance

Requisitos legales

Todos nuestros trabajos incluyen informe técnico avalado por topógrafos certificados para cumplir con:

  • Normas SEDATU
  • Reglamentos de construcción locales
  • Requisitos de seguridad estructural

División/Agrupación de lotes

Proyectos catastrales para modificación legal de propiedades.

División: $10,000 - $24,000 MXN
Agrupación: $7,000 - $14,000 MXN
Trámites incluidos: Avalúo, plano oficial, escrituración

Proceso completo

  1. Estudio técnico: Levantamiento topográfico
  2. Plano oficial: Elaborado según normas catastrales
  3. Avalúo: Valoración de las nuevas propiedades
  4. Trámites notariales: Escrituración y registro
  5. Actualización catastral: Inscripción en RPP

Requisitos legales

  • Superficie mínima según zonificación
  • Accesos independientes para lotes divididos
  • Certificado de libertad de gravámenes
  • Predial al corriente

Beneficios

  • Para divisiones: Maximizar valor de terreno
  • Para agrupaciones: Simplificar administración
  • Asesoría jurídica incluida
  • Gestión ante notario y registro público

Casos especiales

También realizamos:

  • Fusiones de predios
  • Rectificación de colindancias
  • Expedientes para regularización
  • Soluciones a conflictos de límites
